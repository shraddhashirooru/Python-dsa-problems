# Question 4: Print GFG n time:

## Print GFG n times without the loop.

---
### Example:

Input:

5

Output:

GFG GFG GFG GFG GFG

---

## âœ… Python code:

```
class Solution:
    def printGfg(self, n):
        if n==0:
           return
        else:
            print("GFG",end=" ")
            self.printGfg(n-1)
            return n
```
