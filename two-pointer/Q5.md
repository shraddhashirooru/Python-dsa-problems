# Question 5: Reverse the string:

## You are given a string s, and your task is to reverse the string.

### Examples:

Input: s = "Geeks"

Output: "skeeG"

```
        a=list(s)
        n=len(s)
        i=0
        j=n-1
        while i<j:
            a[i],a[j]=a[j],a[i]
            i=i+1
            j=j-1
        return "".join(a)
```
