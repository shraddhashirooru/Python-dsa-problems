# Question 6: Remove consecutive words one by one and return length of arr: 

```
class Solution:
    def removeAdj(self,arr):
        stack=[]
        n=len(arr)
        for i in range(n):
            if stack and arr[i]==stack[-1]:
                 stack.pop()
            else:
                stack.append(arr[i])
        return len(stack)
```
