# Question 27: Next greater element:

```
class Solution:
    def nextGreaterElement(self, nums1: List[int], nums2: List[int]) -> List[int]:
        n=len(nums2)
        stack=[]
        a={}
        for i in range(n-1,-1,-1):
            while stack and stack[-1]<=nums2[i]:
                stack.pop()
            if stack:
                a[nums2[i]]=stack[-1]
            else:
                a[nums2[i]]=-1
            stack.append(nums2[i])
        b=[]
        for i in nums1:
            b.append(a[i])
        return b
```
