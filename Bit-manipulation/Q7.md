# Question 7: Count number of 1 bits:

## Given a positive integer n. Your task is to return the count of set bits.

---
### Examples:
Input: n = 6
Output: 2
Explanation: Binary representation is '110', so the count of the set bit is 2.

```
class Solution:
	def setBits(self, n):
	   count=0
	   while n>0:
	       count=count+n%2
	       n=n//2
	   return count
```
