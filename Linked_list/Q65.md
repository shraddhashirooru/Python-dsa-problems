# Question 65: Pairwise swap of Nodes in a linked list:

## ✅Python code 1:

```
def pairwiseSwap(head):
    prev=None
    current=head
    while current and current.next:
        temp1=current.next
        temp2=current.next.next
        if prev is None:
            current.next=temp1.next
            prev=current
            head=temp1
            temp1.next=prev
            current=temp2
        else:
            prev.next=current.next
            temp1.next=current
            prev=current
            current.next=temp2
            current=temp2
    return head
```
