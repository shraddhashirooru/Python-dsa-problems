# Question 54:  Linked list is palindrome or not:

## Given a head singly linked list of positive integers. The task is to check if the given linked list is palindrome or not.

### Examples:

Input: head: 1 -> 2 -> 1 -> 1 -> 2 -> 1

Output: true

Explanation: The given linked list is 1 -> 2 -> 1 -> 1 -> 2 -> 1 , which is a palindrome and Hence, the output is true.

## ✅Python code 1:

```
class Solution:
    def isPalindrome(self, head):
        s1=[]
        slow=head
        fast=head
        while fast and fast.next:
            s1.append(slow.data)
            slow=slow.next
            fast=fast.next.next
        if not s1:
            return True
        count=0
        while slow:
            a=s1.pop()
            if a!=slow.data:
                if count==0 and slow.next is not None:
                    slow=slow.next
                    if a!=slow.data:
                        return False
                else:
                    return False
            slow=slow.next
        if s1:
            return False
        return True
```

Time: O(n)
Space: O(n)
