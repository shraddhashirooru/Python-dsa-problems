# Question 4: Display linkedList in array:

## Given a singly linked list of integers. The task is to display the linked list.

---
### Example 1:

Input:

LinkedList: 1->2->3->4->5

Output: 1 2 3 4 5

---
## âœ…Python code:

```
def displayList(head):
    arr=[]
    current=head
    while current:
        arr.append(current.data)
        
        current=current.next
    return arr
```
