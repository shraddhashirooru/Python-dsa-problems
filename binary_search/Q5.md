# Question 5: Find whether element x is present in the matrix.

## Given a row-wise sorted 2D matrix mat[][] of size n x m and an integer x, find whether element x is present in the matrix. Note: In a row-wise sorted matrix, each row is sorted in itself, i.e. for any i, j within bounds, mat[i][j] <= mat[i][j+1].

### Examples :

Input: mat[][] = [[3, 4, 9],[2, 5, 6],[9, 25, 27]], x = 9

Output: true

Explanation: 9 is present in the matrix, so the output is true.

class Solution:
    
    #Function to search a given number in row-column sorted matrix.
    def searchRowMatrix(self, mat, x):
        def bin(arr,x):
            n=len(arr)
            i=0
            j=n-1
            while i<=j:
                mid=(i+j)//2
                if arr[mid]==x:
                    return True
                elif arr[mid]<x:
                    i=mid+1
                else:
                    j=mid-1
            return False
        for i in mat:
            if bin(i,x):
                return True
        return False
